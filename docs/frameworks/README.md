# Frameworks

This directory stores evaluation frameworks, metric definitions, and cost models used in the research.

- `cost-taxonomy.md`: direct-cost vs indirect-cost framework for AI problem solving
- `evaluation-framework.md`: quality-constrained total-cost evaluation framework
- `evaluation-objectives.md`: candidate optimization objectives and when to use them
- `logging-schema.md`: episode-level logging schema for benchmark runs
- `cost-aggregation.md`: aggregation and reporting rules for cost metrics
- `failure-taxonomy.md`: shared failure classes for episode analysis
- `policy-design-space.md`: explicit design axes for model, routing, verification, retry, tools, and oversight
- `scenario-cost-assumptions.md`: scenario-specific assumptions for weighted cost comparisons
